# User Management System

A Java application for managing user accounts with full CRUD operations, role assignment, and authentication logic.

## Technologies
- Java 11+
- Object-Oriented Programming

## Features
- Create, read, update, and delete users
- Role assignment (admin, regular user)
- Basic authentication logic
- Console menu interface
- Input validation

## Getting Started
```bash
javac Main.java
java Main
```
